
MR Input 操作例（サンプルキーボード）

キーボードレイアウト

`
(K)　(S)　(T)
(Y)　(A)　(H)
( )　(N)　(R)
`Aは母音キー　タップ、フリックで(a、i、u、e、o)の5母音を入力
基本　母音キーを中央に周囲に子音キーを配置
各子音キーはフリックによって対応濁音、半濁音の子音を入力
各子音のフリック空き方向に他のアルファベットを配置可能
左下の()は空きキーとして任意に機能キーなどを配置できる
小文字は子音のあとに　y　をいれることによって実現
小文字の　っ　は子音のダブルタップによる
いずれもローマ字入力規則のとおり

基本操作
MR Input は「子音 → 母音1キー」の一貫した操作で入力します。

例：
- か → k → 母音キー タップ　a
- き → k → 母音キー 左フリック　i
- く → k → 母音キー 上フリック　u
- け → k → 母音キー　右フリック　e
- こ → k → 母音キー　下フリック　o

もう少し複雑な例
- きょう → k → y → o → u
- しゃ → s → y → a
