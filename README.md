# mr-input
 MR Input — 母音1キー化による最小構造の日本語入力方式

MR Input は、日本語入力の構造を 「母音1キー化」 によって極限まで単純化し、  
スマートフォンでも PC と同じ操作感で入力できるように設計された 最小構造の日本語入力方式 です。

本方式は、ローマ字入力の根本構造を再定義し、  
例外のない一貫したルール と 学習コストの最小化 を両立します。

---

✨ 特徴（Core Features）

- 母音を1キーに集約  
  → a/i/u/e/o を 1 つのキーに統合し、入力構造を最小化。

- スマホでも PC と同じ操作感  
  → キーボードレイアウトに依存しない一貫した入力体験。

- 例外ゼロのルール設計  
  → 入力の特例が存在しない。

- 実装自由度の高さ  
  → キーボードレイアウト、UI、拡張仕様は自由。  
     本リポジトリのサンプルは “公式 UI” として扱うが、  
     他レイアウトの作成は自由。

---

🧩 目的（Philosophy）

MR Input の目的は、  
「日本語入力の構造を美しく最小化し、誰でも実装できるようにすること」。

- コア発明（母音1キー化）は保護しつつ  
- それ以外の UI・レイアウト・実装は自由  
- 商用利用は透明で公平  
- 個人・趣味・研究用途は完全に自由

という “自由と保護のバランス” を重視しています。
