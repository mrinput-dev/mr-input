# mr-input
MR Input
A minimal, vowel‑first Japanese input method.

日本語入力方式を、母音キーから再設計した新しいローマ字入力です。

MR Input keeps the traditional consonant→vowel typing order,  
but restructures the keyboard by placing all vowels on a single central key.  
This central vowel key naturally determines the positions of all consonants,  
creating a shorter, more consistent C→V movement and reducing cognitive load.

This approach offers:

- Consistent key patterns for all Japanese syllables  
- Lower mental overhead compared to standard romaji input  
- A compact, ergonomic keyboard layout  
- Fast, uninterrupted typing flow  

MR Input は、日本語のローマ字入力を 母音キーから再設計した新しい入力方式 です。

従来の「子音 → 母音」というタイピング順序を維持しつつ、  
キーボード上の母音を 中央の1キーに統合することで、  
すべての子音キーの位置が自然に決まり、  
タイピング動作が短く、一貫性が高まり、認知負荷が軽減されます。

---

🧩 特徴

- 母音を1キーに統合した新構造  
- フリック方向をローマ字に持ち込む設計  
- 子音と母音の関係を再構築した方式  
- レイアウト非依存・言語非依存の抽象的な入力原理  
- 実装が容易で、どの IME にも追加可能

MR Input は「高速化のための改造」ではなく、  
入力方式そのものの再設計 を目指した方式です。

---

🔓 自由性について

MR Input は 完全に自由な方式 です。

- 許可不要  
- ロイヤリティ不要  
- 商用利用自由  
- 改変自由  
- 再配布自由  
- 実装自由  

MR は特許や独占権を伴わないため、  
誰でも自由に使えますし、誰の利用も妨げません。

---

🤝 協力関係契約について（任意）

MR Input は自由な方式ですが、  
方式の価値を尊重し、発明者との協力関係を望む方のために、  
任意の「協力関係契約」を用意しています。

これは 方式そのものの対価ではありません。  
方式は自由であり、契約しなくても何の制限もありません。

協力関係契約は、以下のような目的で存在します：

- 発明者を明示的にリスペクトしたい  
- 利用状況を共有したい  
- 方式の発展に協力したい  
- 公式な関係を結びたい  

契約内容の例：

- 利用報告（任意）  
- 発明者の表示（可能な範囲で）  
- 協力関係の維持費（方式の対価ではなく“関係性”の対価）

- 契約をご希望の方はissueにてご連絡ください
- 詳細についてはメールにて連絡いたします

これは OSS の精神と矛盾しない、完全に任意の仕組みです。
